博客部署

首先进入zeabur官网，右上角可切换中英文，登录后点击主页的 现在开始 ，然后新建一个服务，选择地区中如果后面没有按钮那就证明是免费计划，你可以按照需求选择，随后选择GitHub部署，随后选择你的项目即可，如果没有显示项目则点击弹出框右上角按钮 配置GitHub 进行授权，随后弹出框中点击 部署 ，等待部署结束后，在服务的 网络 设置中选择生成域名或者自定域名，完成后即可用网址来访问你的博客。

随后你只需要在修改完博客内容后输入 hexo g 与 hexo d上传到 GitHub ，网页中就能更新内容了，如果上传后等很久都没更新，可以查看终端输出的信息进行调整或者清除浏览器缓存后再尝试。
